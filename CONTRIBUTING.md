# Contributing to StarWar

Thanks for your interest in contributing.

## Development setup

1. Open `StarWar.sln` in Visual Studio.
2. Select `x64` and `Debug`.
3. Build and run `main`.

## Coding guidelines

- Follow the existing C++ style in each file.
- Keep changes minimal and focused.
- Avoid unrelated refactors in gameplay PRs.
- Ensure all modified code builds successfully.

## Pull request checklist

- [ ] Build passes locally
- [ ] Gameplay changes are described in PR notes
- [ ] Docs updated when controls/rules changed
- [ ] No generated binaries or local files committed

## Areas to contribute

- Gameplay balancing and enemy behavior
- Rendering/performance improvements
- Bug fixes in draw2d and collision logic
- Documentation and localization updates
