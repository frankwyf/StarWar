# StarWar（日本語）

モダン C++ と OpenGL/GLFW で作られた、小規模なオープンソース 2D スペースシューティングゲームです。

## 現在のゲーム内容

- マウス照準 + 推進で機体を操作
- 開始前に難易度選択（`Easy / Normal / Hard`）
- 開幕は即戦闘ではなくカウントダウン開始
- アステロイド（分裂あり）との戦闘
- プレイヤーを追尾して撃つ敵機との戦闘
- 4ウェーブごとのボス戦
- ウェーブ進行と武器アップグレード
- ライフ + シールド + 無敵時間 + 位置取りによる生存戦略

## ルールと遊び方

### 目的

できるだけ長く生き残り、ウェーブを突破してボスを倒し、スコアを伸ばします。

### 基本ループ

1. ヘルプ画面で `Enter` を押して開始
2. 回避しながらアステロイド/敵機を撃破
3. 敵機を全滅させると次ウェーブへ
4. `4, 8, 12, ...` ウェーブでボス出現
5. 難易度上昇に対応しながら継続

### 操作

- **1 / 2 / 3**: スタート画面で難易度選択
- **Enter**: ヘルプ画面から開始（カウントダウン付き）
- **マウス**: 照準
- **左クリック**: 射撃
- **W / ↑**: 推進
- **右クリック**: 推進（代替）
- **Space**: パイロットモード切替
- **R**: リスタート
- **Esc**: 終了

### 戦闘と成長

- アステロイド撃破でスコア獲得
- 大型アステロイドは被弾で分裂
- 敵機はプレイヤー方向に射撃
- ボスは高耐久かつ多方向攻撃
- 武器はウェーブ進行で強化：
  - **Lv1**: 単発
  - **Lv2**: 3WAY
  - **Lv3**: 高速連射 + 高火力

### 生存のコツ

- 立ち止まらず常に移動
- 短い推進で弾幕を避ける
- 通常ウェーブは素早く処理して圧を下げる
- 無敵時間を使って有利位置へ戻る

## HUD 表示

- タイトルバー: **Score / Lives / Wave / Weapon / Difficulty**
- 左上: スコア進行バー
- 左上2段目: シールドバー
- 右上: ライフ表示
- ボス機体上: HPバー

## ビルド不要で遊ぶ（推奨）

1. GitHub の **Releases** ページを開く
2. `StarWar-windows-x64.zip` をダウンロード
3. 展開して `StarWar.exe` を実行

## ソースからビルド

### Windows（Visual Studio）

1. `StarWar.sln` を開く
2. `x64` と `Debug` または `Release` を選択
3. `main` プロジェクトをビルド・実行

### クロスプラットフォーム（Premake + Make）

```bash
premake5 gmake2
make config=release_x64
```

## 自動ビルドとリリース

- ワークフローファイル: `.github/workflows/release-windows.yml`
- `main/master` への push ごとに成果物を自動生成
- `v1.0.0` のようなタグで `StarWar-windows-x64.zip` を Releases に自動公開

## プロジェクト構成

- `main/` – ゲーム本体とゲームプレイロジック
- `draw2d/` – ソフトウェアラスタライズと描画プリミティブ
- `support/` – 実行時/設定ユーティリティ
- `vmlib/` – 数学ライブラリ（`Vec2f`, `Mat22f`）
- `lines-test/`, `triangles-test/` – ユニットテスト
- `blit-benchmark/`, `lines-benchmark/` – ベンチマーク
- `third_party/` – サードパーティ依存
